LightweightMatrixCSharp

Данный проект представляет собой набор классов для работы с матрицами разного размера. Основной фишкой проекта является реализация алгоритма Штрассена для работы с матрицами большой размерности.

Да, подобных наработок можно найти сотни, если не тысячи в сети. Данный проект предоставляет возможность онаознакомиться со стилем написания кода и профессионализмом автора.
